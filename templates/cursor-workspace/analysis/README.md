# Analysis

Persistent working artifacts for features, bugs, refactors and technical decisions.

Use this folder to keep durable context across sessions instead of relying only on chat history.

## Structure

```text
analysis/
├─ README.md
├─ shared/
│  └─ review-guidelines.md
└─ features/
   └─ <feature-slug>/
      ├─ feature-manifest.md
      ├─ user-story.md
      ├─ analysis.md
      ├─ implementation-notes.md
      └─ test-checklist.md
```
